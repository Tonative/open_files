# African Storytelling Dataset (Preview)

A multilingual speech dataset of African stories featuring natural narration in four languages — **Yoruba**, **Igbo**, **Hausa**, and **Dholuo**. Each recording is 5–10 minutes long, capturing the natural prosody, rhythm, and oral structures of African narrative speech.

## What's in this preview

| File | Language | Audio | Transcript | Duration | Genre |
|------|----------|:-----:|:----------:|----------:|----------|
| `Yoruba_id_aa` | Yoruba | ✅ | - | ~5–10 min | Contemporary Narrative |
| `Igbo_id_aa` | Igbo | - | ✅ | ~5–10 min | Contemporary Narrative |
| `Hausa_id_aa` | Hausa | - | ✅ | ~5–10 min | Contemporary Narrative |
| `Hausa_id_ab` | Hausa | ✅ | ✅ | ~5–10 min | Contemporary Narrative |
| `Dholuo_id_aa` | Dholuo | ✅ | ✅ | ~5–10 min | Folktale |

## Languages

| Language | ISO 639 | Family | Region |
|----------|---------|--------|--------|
| Yoruba | yo | Niger-Congo | Nigeria, Benin, Togo |
| Igbo | ig | Niger-Congo | Nigeria |
| Hausa | ha | Afroasiatic | Nigeria, Niger, Ghana |
| Dholuo | luo | Nilotic | Kenya, Tanzania |

## Use cases

- **Automatic Speech Recognition (ASR)** for low-resource African languages
- **Text-to-Speech (TTS)** training with natural narrative prosody
- **Speech-text alignment** research
- **Linguistic analysis** of oral storytelling structures
- **Language preservation** and documentation
